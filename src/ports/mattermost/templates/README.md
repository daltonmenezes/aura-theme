# Installation
In Mattermost App
1. Open **Settings**
2. Select **Display** from Sidebar
3. Select **Theme** > **Edit**
4. Select **Custom Themes**
5. Copy the content from [aura-dark.json](https://raw.githubusercontent.com/{{ author.username }}/{{ slug }}/main/packages/mattermost/aura-dark.json)
