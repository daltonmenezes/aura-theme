# Installation

1. Save [`aura-theme.json`](aura-theme.json) to `~/.config/tilix/schemes` folder without txt extension that the browser will infer
2. Open a new Tilix window and select the Aura Theme on `Preferences -> Profile -> Color`.

<br/>
Done! ✨ 🎉
